# Weather App

A simple Weather App built using HTML, CSS and JavaScript.  
It fetches live weather data using the OpenWeatherMap API.

## Features
- Search weather by city
- Displays temperature and weather condition
- Weather icons
- Enter key search
- Modern glass UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
- OpenWeatherMap API
